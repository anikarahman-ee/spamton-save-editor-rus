# 🚀 Быстрый старт - Десктопное приложение

## Способ 1: Запуск из исходников (РЕКОМЕНДУЕТСЯ для начала)

### Шаг 1: Установите Node.js
1. Скачайте Node.js с официального сайта: https://nodejs.org/
2. Установите (выбирайте LTS версию)
3. Перезагрузите компьютер после установки

### Шаг 2: Запустите приложение
1. Откройте папку `desktop`
2. Запустите файл **`START.bat`** (двойной клик)
3. Подождите установки зависимостей (происходит один раз)
4. Приложение откроется автоматически!

**Вот и всё!** Приложение готово к работе.

---

## Способ 2: Сборка исполняемого файла

Если хотите создать `.exe` файл для запуска без Node.js:

### Шаг 1: Установите зависимости
1. Откройте папку `desktop`
2. Запустите **`BUILD.bat`**
3. Дождитесь завершения сборки (5-10 минут)

### Шаг 2: Найдите готовое приложение
Файлы будут в папке `desktop/build/`:
- **Установщик**: `Deltarune Save Editor Setup.exe`
- **Портативная версия**: папка `win-unpacked/`

---

## Первое использование

1. **Запустите приложение**
2. **Нажмите "🔄 Обновить список сейвов"**
   - Приложение автоматически найдет сохранения Deltarune
   - Они обычно находятся в `C:\Users\ВашеИмя\AppData\Local\DELTARUNE`

3. **Выберите сохранение** из списка слева
   - Кликните на файл (например, `filech1_0`)

4. **Выберите главу** для редактирования
   - Кликните на одну из глав (01, 02, 03, 04)

5. **Редактируйте параметры** в центре
   - Изменяйте характеристики, предметы, флаги событий

6. **Сохраните изменения**
   - Нажмите "💾 Сохранить изменения"
   - Автоматически создастся резервная копия!

## ❓ Проблемы?

### "Node.js не найден"
→ Установите Node.js с https://nodejs.org/ и перезагрузите ПК

### "Сохранения не найдены"
→ Убедитесь что Deltarune установлена и запускалась хотя бы раз

### "Ошибка при сохранении"
→ Закройте Deltarune перед редактированием сейва

## 💡 Полезное

### Открыть папку с сохранениями вручную
1. Нажмите `Win + R`
2. Введите: `%LOCALAPPDATA%\DELTARUNE`
3. Нажмите Enter

### Восстановить из бэкапа
1. Найдите файл с расширением `.backup_дата-время`
2. Удалите основной файл (например, `filech1_0`)
3. Переименуйте бэкап, убрав `.backup_дата-время`
