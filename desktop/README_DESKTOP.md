# Deltarune Save Editor - Десктопная версия

**Редактор сохранений Deltarune для Windows с автоматическим поиском и изменением файлов**

## 🎮 Особенности

- **Автоматический поиск сохранений** - находит файлы Deltarune в `%LOCALAPPDATA%\DELTARUNE`
- **Прямое редактирование** - изменяет сейвы напрямую без необходимости загрузки/выгрузки
- **Автоматические бэкапы** - создает резервные копии перед изменениями
- **Идентичный интерфейс** - использует тот же редактор что и веб-версия
- **Поддержка всех глав** - Главы 1, 2, 3, 4 и демо-версии

## 📦 Установка

### Вариант 1: Запуск из исходников (для разработчиков)

1. **Установите Node.js** (если еще не установлен)
   - Скачайте с https://nodejs.org/
   - Рекомендуется LTS версия

2. **Установите зависимости:**
   ```bash
   cd desktop
   npm install
   ```

3. **Запустите приложение:**
   ```bash
   npm start
   ```

### Вариант 2: Сборка исполняемого файла

1. **Установите зависимости** (если еще не установлено):
   ```bash
   cd desktop
   npm install
   ```

2. **Соберите приложение:**
   ```bash
   npm run build-win
   ```

3. **Найдите готовое приложение:**
   - Установщик: `desktop/build/Deltarune Save Editor Setup.exe`
   - Портативная версия: `desktop/build/Deltarune Save Editor.exe`

## 🚀 Использование

### Первый запуск

1. Запустите приложение
2. Нажмите **"Обновить список сейвов"** для автоматического поиска
3. Выберите нужное сохранение из списка слева
4. Выберите главу для редактирования
5. Редактируйте параметры в центральной области
6. Нажмите **"💾 Сохранить изменения"** для применения

### Основные функции

#### 🔄 Обновить список сейвов
Автоматически сканирует папку `%LOCALAPPDATA%\DELTARUNE` и находит все файлы сохранений.

#### 📁 Открыть папку с сейвами
Открывает папку Deltarune в проводнике Windows для ручного управления файлами.

#### 📂 Выбрать файл вручную
Позволяет выбрать сохранение из произвольной папки (например, для импорта).

#### 💾 Сохранить изменения
Применяет изменения напрямую к файлу сохранения. Автоматически создается резервная копия.

### Резервные копии

При каждом сохранении автоматически создается файл `.backup` с датой и временем.
Пример: `filech1_0.backup_2026-03-09T14-30-45`

Для восстановления просто переименуйте бэкап, убрав расширение `.backup_*`.

## 🐛 Решение проблем

### Приложение не находит сохранения

1. Убедитесь что Deltarune установлена и запускалась хотя бы раз
2. Проверьте папку вручную: 
   - Нажмите `Win+R`
   - Введите: `%LOCALAPPDATA%\DELTARUNE`
   - Должны быть файлы `filech1_0`, `filech2_0` и т.д.

### Ошибка при сохранении

1. Проверьте права доступа к папке Deltarune
2. Закройте игру перед редактированием сейва
3. Попробуйте запустить приложение от имени администратора

### Редактор не загружается

1. Убедитесь что все файлы проекта на месте (особенно папки `js/`, `css/`)
2. Проверьте консоль разработчика (F12) на наличие ошибок

## 🔒 Безопасность

- Все операции с файлами происходят локально
- Нет подключения к интернету (кроме загрузки jQuery с CDN для редактора)
- Автоматические бэкапы предотвращают потерю данных
- Используется Context Isolation для безопасности Electron
