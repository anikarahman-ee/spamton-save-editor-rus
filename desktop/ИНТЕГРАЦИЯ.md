# Интеграция десктопного режима с существующими HTML страницами

## Что нужно сделать

Для работы десктопного приложения необходимо добавить скрипт интеграции на страницы редактора глав.

## Автоматическая интеграция (РЕКОМЕНДУЕТСЯ)

Просто добавьте эту строку в `<head>` каждого файла редактора глав:

```html
<script src="desktop/desktop-integration.js"></script>
```

### Файлы для модификации:
- `deltarune1.html`
- `deltarune2.html`
- `deltarune3.html`
- `deltarune4.html`
- `deltarune1Demo.html`
- `deltarune2Demo.html`

### Пример добавления:

**БЫЛО:**
```html
<head>
    <meta charset="UTF-8">
    <title>Deltarune Chapter 1 Editor</title>
    
    <!-- Существующие скрипты -->
    <script src="js/editorCore.js"></script>
    ...
</head>
```

**СТАЛО:**
```html
<head>
    <meta charset="UTF-8">
    <title>Deltarune Chapter 1 Editor</title>
    
    <!-- НОВОЕ: Интеграция с десктопным приложением -->
    <script src="desktop/desktop-integration.js"></script>
    
    <!-- Существующие скрипты -->
    <script src="js/editorCore.js"></script>
    ...
</head>
```

## Что делает скрипт интеграции?

1. **Автоматически определяет** запущено ли приложение в десктопном режиме
2. **Принимает данные сейва** от родительского окна
3. **Автоматически загружает** сейв в редактор
4. **Перехватывает сохранение** и отправляет данные напрямую в файловую систему
5. **Скрывает ненужные элементы** (облачные функции, навигацию)
6. **Добавляет кнопку прямого сохранения** для удобства

## Проверка работы

После добавления скрипта:

1. Откройте консоль браузера (F12) при работе в десктопном приложении
2. Вы должны увидеть сообщение:
   ```
   [Desktop Integration] Режим десктопного приложения активирован
   ```

3. При загрузке сейва появится:
   ```
   [Desktop Integration] Получены данные сейва: filech1_0
   [Desktop Integration] Сейв загружен в редактор
   ```

## Обратная совместимость

✅ Скрипт **полностью безопасен** для веб-версии!

- В обычном браузере скрипт обнаружит отсутствие десктопного режима
- Редактор продолжит работать как обычно
- Никакой функционал не нарушится

## Альтернатива: ручной способ

Если не хотите изменять существующие файлы, можете:

1. Создать копии HTML файлов специально для десктопа
2. Например: `deltarune1-desktop.html`
3. Изменить загрузку в `desktop-app.js` чтобы использовать эти файлы

Но это **не рекомендуется**, т.к. придется поддерживать два набора файлов.

## 🔧 Автоматизация

Для автоматического добавления скрипта во все файлы можно использовать скрипт:

```javascript
// add-integration.js - запустить через: node add-integration.js
const fs = require('fs');
const files = [
    'deltarune1.html',
    'deltarune2.html',
    'deltarune3.html',
    'deltarune4.html',
    'deltarune1Demo.html',
    'deltarune2Demo.html'
];

const scriptTag = '    <script src="desktop/desktop-integration.js"></script>\n';

files.forEach(file => {
    let content = fs.readFileSync(file, 'utf8');
    
    // Проверяем, не добавлен ли уже
    if (content.includes('desktop-integration.js')) {
        console.log(`✓ ${file} - уже содержит интеграцию`);
        return;
    }
    
    // Добавляем после <head>
    content = content.replace('<head>', '<head>\n' + scriptTag);
    
    fs.writeFileSync(file, content);
    console.log(`✓ ${file} - интеграция добавлена`);
});

console.log('\nГотово!');
```

Сохраните этот код в файл `add-integration.js` в корне проекта и запустите:
```bash
node add-integration.js
```
