# Примечание об иконке

## 📝 Иконка приложения

Для полноценной работы приложению нужна иконка.

### Что нужно сделать:

1. **Создайте или найдите иконку** в формате PNG (256x256 или больше)
2. **Сохраните её в папке `desktop/`** под именем `icon.png`
3. **Для Windows также создайте** `icon.ico` (можно конвертировать онлайн)

### Рекомендации по дизайну:

- Используйте тематику Deltarune или Spamton
- Размер: 256x256 px (PNG), 256x256 px (ICO)
- Прозрачный фон
- Яркие цвета (фиолетовый #9b59f4)

### Инструменты для создания:

**Онлайн конвертеры PNG → ICO:**
- https://convertio.co/ru/png-ico/
- https://www.icoconverter.com/
- https://favicon.io/

**Редакторы:**
- Paint.NET (Windows)
- GIMP (кроссплатформа)
- Photoshop (профессиональный)

### Альтернатива:

Если иконки нет, приложение будет использовать стандартную иконку Electron.

### Использование существующей графики:

Можно взять изображение из папки `images/` проекта и адаптировать его.

---

**После добавления иконки пересоберите приложение:**
```bash
npm run build-win
```
